\doxysubsection{Detailed Description}
JPEG-\/like decompression front-\/end for color assets. 

The class reads the metadata produced by \doxylink{classcCompressionCouleur}{c\+Compression\+Couleur}, restores each channel to the spatial domain, performs chroma upsampling, and converts the YCb\+Cr planes back to interleaved RGB pixels.
