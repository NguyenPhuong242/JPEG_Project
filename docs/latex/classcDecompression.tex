\doxysubsection{Detailed Description}
High-\/level JPEG decompression helper. 

The class mirrors the compression interface but focuses solely on reconstructing images from the Huffman-\/compressed bitstream produced by \doxylink{classcCompression}{c\+Compression}. It performs Huffman decoding, inverse RLE, dequantization, and IDCT to rebuild the spatial domain image.
