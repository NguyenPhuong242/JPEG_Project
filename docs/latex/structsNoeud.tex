\doxysubsection{Detailed Description}
Node used by the pedagogic Huffman tree implementation. 

Contains a symbol (one byte) and its frequency. Left/right pointers form the binary tree used to derive codes. This is a simple, non-\/owning structure; \doxylink{classcHuffman}{c\+Huffman} is responsible for deleting allocated nodes.
