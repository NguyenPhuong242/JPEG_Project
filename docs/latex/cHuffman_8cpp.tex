\doxysubsection{Detailed Description}
Simple Huffman tree implementation used by the pedagogic compressor. 

The implementation builds a binary tree from symbol frequencies and provides a traversal helper to print codes. Memory ownership\+: this class owns the copy of the input trame and deletes tree nodes on destruction.
