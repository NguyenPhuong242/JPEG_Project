Educational grayscale JPEG pipeline that showcases every major step of the codec\+: block extraction, 8x8 DCT, quantization, zig-\/zag + run-\/length encoding, Huffman entropy coding, and a CLI that can compress and decompress sample images.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md1}

\begin{DoxyItemize}
\item Baseline JPEG-\/style pipeline implemented in modern C++.
\item Modular headers under {\ttfamily include/} for reuse in other exercises.
\item CLI ({\ttfamily jpeg\+\_\+cli}) able to compress, decompress, or run a full round trip.
\item Doxygen documentation with class and file overviews.
\item Lightweight test binaries covering DCT, quantization, Huffman, and RLE pieces.
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Project Layout}{Project Layout}}\label{index_autotoc_md2}
