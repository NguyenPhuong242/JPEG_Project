\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{1. Overview}{1. Overview}}\label{index_autotoc_md2}
This project provides a modular and pedagogical implementation of the essential steps of a JPEG-\/style compression/decompression chain. It is designed to demonstrate\+:


\begin{DoxyItemize}
\item 8×8 block splitting
\item DCT (Discrete Cosine Transform) / IDCT
\item Quantization
\item RLE (Run-\/\+Length Encoding)
\item Huffman entropy coding
\item Optional YCb\+Cr color pipeline with subsampling
\end{DoxyItemize}

{\bfseries{Note\+:}} The produced {\ttfamily \texorpdfstring{$\ast$}{*}.huff} files are a custom, transparent format for educational purposes and are not standard JPEG files.

\DoxyHorRuler{0}
\hypertarget{index_autotoc_md4}{}\doxysection{\texorpdfstring{2. Features}{2. Features}}\label{index_autotoc_md4}

\begin{DoxyItemize}
\item {\bfseries{Grayscale pipeline\+:}} Compression of ASCII pixel maps to custom bitstreams.
\item {\bfseries{Color pipeline\+:}} Full support for PPM (P6) images converted to Y/\+Cb/\+Cr streams.
\item {\bfseries{Chroma Subsampling\+:}} Supports 4\+:4\+:4 (no subsampling), 4\+:2\+:2, and 4\+:2\+:0.
\item {\bfseries{Modular Design\+:}} Distinct modules for DCT, Quantization, RLE, and Huffman.
\item {\bfseries{Analysis Tools\+:}} Frequency histogram generation and unit tests.
\item {\bfseries{Documentation\+:}} Doxygen-\/ready comments.
\end{DoxyItemize}

\hypertarget{index_autotoc_md26}{}\doxysection{\texorpdfstring{7. Important Notes}{7. Important Notes}}\label{index_autotoc_md26}

\begin{DoxyEnumerate}
\item {\bfseries{File Format\+:}} The {\ttfamily .huff} format used here is {\bfseries{custom}}. It is designed to make the RLE and Huffman encoding steps transparent for debugging and learning. It is {\bfseries{not}} compatible with standard {\ttfamily .jpg} viewers.
\item {\bfseries{Quality Factor\+:}} The quality argument (1–100) controls the quantization matrix.
\begin{DoxyItemize}
\item {\bfseries{Low value\+:}} High compression, lower quality (more data loss).
\item {\bfseries{High value\+:}} Low compression, higher quality. 
\end{DoxyItemize}
\end{DoxyEnumerate}
