\doxysubsection{Detailed Description}
Extends \doxylink{classc_compression}{c\+Compression} to handle color image compression (PPM format). 

This class manages the pipeline for compressing and decompressing P6 (binary) PPM color images. It handles reading the PPM file, converting from RGB to YCb\+Cr color space, performing chroma subsampling (4\+:4\+:4, 4\+:2\+:2, 4\+:2\+:0), and then uses the base \doxylink{classc_compression}{c\+Compression} functionality to compress each color component (Y, Cb, Cr) into a separate file.
