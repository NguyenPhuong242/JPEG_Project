\doxysubsection{Detailed Description}
Functor for comparing two \doxylink{structs_noeud}{s\+Noeud} pointers based on frequency. 

This is used by the {\ttfamily std\+::priority\+\_\+queue} to build the Huffman tree. It ensures that nodes with lower frequencies have higher priority, which is essential for the Huffman algorithm.
