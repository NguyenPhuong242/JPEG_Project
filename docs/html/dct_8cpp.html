void Calcul_DCT_Block ( int ** Bloc8x8,
double ** DCT_Img )

Computes the forward 2D-DCT of an 8x8 block.

This function transforms an 8x8 block from the spatial domain to the frequency domain.

Note
"Calcul" is French for "Calculation".
Parameters
[in]Bloc8x8An 8x8 input block of pixel data (typically level-shifted by -128).
[out]DCT_ImgAn 8x8 output block to be filled with the resulting DCT coefficients.
