enum class jpeg::ChromaSubsampling : std::uint32_t
strong

Chroma subsampling patterns supported by the color codec.

Values match the classic JPEG sampling notation where the first digit is the luma sampling rate and the remaining digits describe the chroma decimation ratio.

Enumerator
Sampling444 

No subsampling (full resolution chroma).

Sampling422 

Horizontal chroma downsampling by 2.

Sampling420 

Horizontal and vertical downsampling by 2.

Sampling411 

Horizontal chroma downsampling by 4.
